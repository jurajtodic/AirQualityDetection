# AirQualityDetection
Time series analysis for indoor air quality prediction and event detection, currently in development

The current (of many) goals is to create a classification model that could estimate whether the door was opened or not based on the sensor data. In other words, when there is a change in temp or conc. pollutant, is it related to the opening of the door or not.
